# Битва за Москву — сайт с 5 картами и точками военных частей

## Запуск

```powershell
cd путь\к\папке\moscow-battle-timeline-maps-units
pip install -r requirements.txt
py -m uvicorn main:app --reload
```

Открыть:

```text
http://127.0.0.1:8000
```

## Где менять карты и этапы

Файл:

```text
data/timeline.json
```

Главное поле карты:

```json
"mapImage": "/static/maps/01_october_13_18.jpg"
```

## Как добавить военную часть

В нужном этапе найди массив:

```json
"units": []
```

И добавь туда объект:

```json
{
  "name": "16-я армия",
  "short": "16А",
  "color": "blue",
  "x": 60,
  "y": 45,
  "description": "Краткое описание части и её роли на этом этапе."
}
```

## Поля военной части

- `name` — полное название.
- `short` — короткая подпись на точке.
- `color` — цвет точки: `"blue"` или `"red"`.
- `x` — положение по горизонтали в процентах от ширины карты.
- `y` — положение по вертикали в процентах от высоты карты.
- `description` — текст при клике на точку.

## Как работают координаты

```text
x: 0    левый край карты
x: 100  правый край карты

y: 0    верх карты
y: 100  низ карты
```

Пример:

```json
"x": 50,
"y": 50
```

Это центр карты.

## Как убрать все части с этапа

Оставь пустой массив:

```json
"units": []
```

## Цвета

Для советских частей удобно использовать:

```json
"color": "blue"
```

Для немецких частей:

```json
"color": "red"
```
